# SRinis: 내일배움캠프 1주차 팀원 소개 앱 프로젝트
SRinis는 내일배움캠프 1주차 팀원을 소개하는 간단한 앱 프로젝트입니다. 이 앱은 팀원들의 정보와 간단한 소개를 제공하여 팀원 간의 소통과 협업을 촉진합니다.
