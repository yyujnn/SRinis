# 🧑🏻‍🏫 SRinis: 내일배움캠프 1주차 팀 소개 앱 프로젝트


- 👬 TEAM 스린이들 (iOS_3기 2조)
- `swift` + `어린이들`

<br>

```
💡 프로젝트 주제 : 팀 소개 앱 만들기

  - iOS 개발자를 목표로 함께 Swift를 공부하는 ‘스린이들’ 소개

  - 학교 컨셉으로 구성한 팀 소개 앱

  - 팀원들의 정보와 간단한 소개를 제공하여 팀원 간의 소통과 협업을 촉진

```

## 📌 Features

**1) LaunchView**
- (SwiftUI애니메이션)

**2) Main page**
- Bottom Tab Bar Controller
- 팀 소개,팀원 소개
  
**3) 팀 소개 pagee**
- ScrollView 활용 
- 역할 및 목표, 협력소개, 성과 및 소감, 미래비전

**4) 팀원 소개 page**
- CollectionView 활용(버튼 활용해 ui구성 변경)
- 멤버 data 연결
- 카드 view
- 프로필 리스트 view
 
**5) 프로필 상세 page**
- SwiftUi 활용
- 각 팀원의 상세 정보
- 깃허브, 블로그 링크 연결
 
**6)Quiz page**
- 각 멤버 당 2문제씩 제공
- 오답 및 정답 Alert 표시

## 📚 Feel
#### 🔧 기초 지식 부족

- 팀원들의 요구사항을 이해하는 데 어려움이 있었습니다.
- 경험이 가장 빠른 학습이라는 말처럼 혼자서의 학습 시간 및 팀원들의 이해로 고민하며 성장했습니다.

#### ⚙️ 개발 경험

- 프로젝트를 목표한 대로 완료하여 자신감을 얻었습니다.
- 처음으로 프로그램에 직접 참여하여 페이지를 만들어본 것은 소중한 경험이었습니다.
- 첫 프로젝트로 간단한 앱을 만들어서 뿌듯하고 성장했다고 느꼈습니다.

#### 💪  협업 능력

- 함께 고민하고 해결해 나가는 과정을 통해 협업의 중요성을 깨달았습니다.
- 진행사항을 서로 공유하고 계획을 짜면서 효율적으로 프로젝트를 진행할 수 있었습니다.
- 서로의 아이디어를 공유하고 함께 진행하면서 집단지성의 힘을 느낄 수 있었습니다.
- 모르는 부분을 서로 알려주고 함께 해결할 수 있어서 감사했습니다.
- 단기간 내에 프로젝트를 완료하면서 새로운 사람들과의 소통과 협업을 경험하면서 성장했다고 생각합니다.
